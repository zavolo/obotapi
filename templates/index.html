Документация API

Сервер поддерживает стандартные методы Telegram Bot API. Используйте следующий формат для запросов:

https://api.opengra.me/bot<token>/<method>

Полная документация доступна по адресу:

Открыть документацию

Поддерживаемые методы

  • getMe - получить информацию о боте
  • sendMessage - отправить сообщение (GET/POST)
  • deleteMessage - удалить сообщение
  • editMessageText - редактировать текст сообщения
  • getUpdates - получить обновления (long polling)
  • answerCallbackQuery - ответить на callback запрос

Пример использования

GET запрос:

curl "https://api.opengra.me/bot123:ABC/sendMessage?chat_id=1488&text=Те беззаботные года юности, пора, думал, не повзрослею, и будет так всегда"

POST запрос (JSON):

curl -X POST "https://api.opengra.me/bot123:ABC/sendMessage" \
  -H "Content-Type: application/json" \
  -d '{"chat_id": 1488, "text": "Может я и балуюсь, но не так уж жестко"}'
